# Gallumbos Ciruelo 🩲

Sitio web de una marca ficticia de ropa interior, desarrollado como proyecto de FP Informática.

## Tecnologías usadas
- HTML5
- CSS3 (Grid, Flexbox, variables CSS)

## Características
- Navegación fija con blur
- Catálogo de productos
- Formulario de contacto
- Diseño responsive
